Nextedy GANTT 2.4

We have just released a new version of our Planning solution for Polarion ALM.

  • The main improvement (in addition to bug fixes) is the out of the box support for comparing current schedule with historical status.
  • In the previous versions you had to use Nextedy API to achieve the same results.
  • Now, you can compare the schedule with:
    • relative (T-Minus) dates
    • any date you pick on calendar
    • or connect the Widget Parameter with Page Parameters to let the user pick the date
  • Expand All / Collapse All - toolbar actions has ben added to support navigation through the large plans

To learn more, read the support article  or download the new version to give it a try.
